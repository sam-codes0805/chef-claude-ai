# Chef Claude (AI Recipe Generator)

A web application that generates recipes based on user-provided ingredients using AI.

## Features
- Generate recipes using AI API
- Dynamic user input handling
- Responsive UI with Tailwind CSS

## Tech Stack
- React.js
- Tailwind CSS
- AI API Integration

## Live Demo
https://chef-claude-demo.netlify.app

## How to Run
1. Clone the repository
2. add your GEMINI-AI Api key to .env file
4. Run npm install
5. Run npm start
